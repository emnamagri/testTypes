What is CI/CD ?

Continuous Integration and Continuous Delivery (CI/CD) are terms used to describe a process where multiple changes are made to a codebase simultaneously.

  • CI : Continuous integration (CI) is the practice of automating the integration of code changes from multiple contributors into a single software project.
  • CD : Continuous delivery is an extension of continuous integration since it automatically deploys all code changes to a testing and/or production environment after the build stage.
